# ITERA | Automatización con Inteligencia Artificial

![ITERA Logo](globe.png)

## 🚀 Revoluciona tu negocio con IA

ITERA es una landing page futurista diseñada para una agencia de automatización con Inteligencia Artificial. El proyecto destaca por su estética premium, efectos visuales avanzados con Three.js y una experiencia de usuario optimizada.

### ✨ Características Principales

- **Diseño Futurista**: Estética "Dark Mode" con acentos neón y glassmorphism.
- **Gráficos 3D**: Fondo interactivo con una red neuronal dinámica usando **Three.js**.
- **Calculadora de ROI**: Herramienta interactiva para que los clientes potenciales estimen sus ahorros al automatizar procesos.
- **Asistente Virtual**: Widget de chat integrado para captación de leads.
- **Totalmente Responsivo**: Optimizado para móviles, tablets y escritorio.
- **SEO Optimizado**: Estructura semántica y meta-tags configurados.

### 🛠️ Tecnologías Utilizadas

- **HTML5 & CSS3**: Estructura y estilos (Vanilla CSS).
- **JavaScript (ES6+)**: Lógica de interacción y animaciones.
- **Three.js**: Renderizado de red neuronal 3D en tiempo real.
- **Lucide Icons**: Iconografía moderna y minimalista.

### 📂 Estructura del Proyecto

- `index.html`: Página principal.
- `privacidad.html`: Política de privacidad.
- `style.css`: Estilos globales y sistema de diseño.
- `script.js`: Lógica del sitio, calculadora y Three.js.
- `globe.png`: Assets visuales.

### Despliegue recomendado en EasyPanel

Este proyecto incluye `Dockerfile`, `nginx.conf` y configuracion runtime para desplegar la landing como una App Docker en EasyPanel.

Consulta `EASYPANEL_DEPLOY.md` para conectar:
- landing web,
- n8n,
- Supabase,
- Google Sheets,
- Google Calendar,
- OpenAI.

### 🚀 Despliegue en GitHub Pages

Para desplegar este proyecto en GitHub Pages:
1. Sube este repositorio a GitHub.
2. Ve a **Settings** > **Pages**.
3. En **Branch**, selecciona `main` y la carpeta `/(root)`.
4. ¡Listo! Tu página estará en línea en unos minutos.
